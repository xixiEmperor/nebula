# 侧边栏
## 顶部
1. 左边头像加用户昵称
点击之后会出现一个弹框，如下图
![1758719031898](image/项目页/1758719031898.png)
- 头像 + 个人信息 + 邮箱
- Theme：可切换主题
- Create替换为Account：个人中心，点击后跳转至个人中心
- Log out：登出

2. 右边是一个铃铛，可以用来接收协作者的消息（Websocket）

## 搜索框
点击后会出现一个下拉框，可以搜索Recents里面的项目

## 选项菜单
1. Recents：可以看到自己最近的项目，用户在创建项目的时候可以选择Private或者Public
2. Shared：自己和其他人共创的项目
3. Community：可以看到所有注册的用户的Public项目

## 团队下拉菜单
1. 草稿
2. 回收站

# 二级路由：项目卡片组
二级路由里面只展示所有相应类型的项目卡片，卡片支持预览，编辑和删除
